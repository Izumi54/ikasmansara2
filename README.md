# IKA SMANSARA Web App

Aplikasi Web Portal Ikatan Alumni SMAN 1 Jepara (IKA SMANSARA). Aplikasi ini bertujuan untuk menghubungkan kembali para alumni, memfasilitasi kolaborasi, serta menyediakan informasi terkini seputar sekolah dan kegiatan alumni.
